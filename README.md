# Servidor con Node y Mysql

## Description

Este proyecto es una prueba de concepto para la creación de un backend de productos y categorías, utilizando NodeJS y Mysql.

## Endpoints

### Productos

- GET /api/product
- GET /api/product/:id
- GET /api/product/term/:term

### Categorías

- GET /api/category
- GET /api/product/category/:id

## Installation

```bash
 npm install
```
